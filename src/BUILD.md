## Quick Build Commands

**Windows:**
```cmd
build.bat
```

**Linux/Unix/macOS:**
```bash
chmod +x build.sh
./build.sh
```

**Universal (any platform):**
```bash
chmod +x build-universal.sh
./build-universal.sh
```

**Linux/Unix/macOS:**
```bash
./scheduler
```
